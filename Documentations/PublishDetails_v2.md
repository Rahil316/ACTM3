## Full Description

You already know what design system you want. Maybe it's Material, Shadcn, Tailwind, or something custom. Getting it into Figma variables is the problem — the mapping, the naming, the dark mode, the contrast checks, the alias chains. It takes days. It breaks when you touch it. And the moment a brand color changes, you're back to square one.

Token Wand takes care of all of it.

---

### Set it up once. Update in seconds.

Tell Token Wand your colors, your semantic roles, and your themes. Click Run. It builds the entire Figma variable system — tonal scales, semantic tokens, alias chains, dark mode modes, source constants — and publishes it directly into your file.

When a brand color changes, you update one hex value and click Run again. Every scale, every token, every mode updates automatically. Nothing breaks.

---

### Accessibility is built into the system, not checked after

Every token in your system is solved to a WCAG contrast target you set. Not checked, not warned about — solved. Token Wand finds the exact color that meets your accessibility requirement and writes that as the variable value. If you use it right, a low-contrast warning is not something you will ever see.

Light theme, dark theme, surface layers, text on buttons — each role contrasts against the right background, not just the page background. The math is done. The variables are correct.

---

### Start from what you already know

Load a preset that matches your direction — Material Design 3, Tailwind CSS, Shadcn/Radix, IBM Carbon, Shopify Polaris, Apple HIG — and it's already configured as a full Figma variable system. Swap your brand colors. Click Run.

Unlike copying a community library, you own the system completely. Change the number of steps, rename a role, add a theme, tighten a contrast target. Everything regenerates. Nothing needs a manual audit.

---

### Hand-off ready out of the box

Developers get what they need directly from the plugin — CSS custom properties, SCSS, Tailwind config, DTCG JSON, Style Dictionary, Swift, Android, React Native. No copy-paste, no spreadsheet, no translation layer.

What's in Figma is what ships.

---

### Everything updates together

Because the whole system is generated from your configuration, it stays consistent. Rename a color, reorder roles, add a dark mode — the plugin tracks every change by identity, not position. Existing variables rename in place. Component references never break.
